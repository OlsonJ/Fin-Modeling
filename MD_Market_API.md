|Resources|Descriptor|
| -- | -- |
|`https://site.financialmodelingprep.com/developer/docs`| Free Stock Market API and Financial Statements API |

|Search Types|Descriptor|
| -- | -- |
| `Company Name Search API` | Search for tickers and exchange details for equity securities and ETFs with the FMP Name Search API. Endpoint for retrieving tickers knowing full/partial company or asset name but not the symbol identifier. |
| `Search CIK API` | Retrieve Central Index Key (CIK) for publicly traded companies with the FMP CIK API. Access unique identifiers for SEC filings and regulatory documents for a streamlined compliance and financial analysis process. |
| `CUSIP API` | search and retrieve financial securities information by CUSIP number using the FMP CUSIP API. Find company name, stock symbol, and market capitalization associated with the CUSIP. |
| `Search ISIN API` | search and retrieve the International Securities Identification Number (ISIN) for financial securities using the FMP ISIN API. Find key details such as company name, stock symbol, and market capitalization associated with the ISIN. |

There are a myriad of APIs, which can be found at the link at the top.
